class app.api.engines.ContentEngine.ContentEngine¶

Bases: object

train()¶

Train the engine.

Create a TF-IDF matrix of unigrams, bigrams, and trigrams for each product.

Then similarity is computed between all products using SciKit Cosine Similarity.

Iterate through each item's similar items and store the 50 most-similar. Similarities and their scores are stored in database as a sorted set, with one set for each item.

Returns:None
train_item(item_id)¶

Train the engine for a given item.

Create a TF-IDF matrix of unigrams, bigrams, and trigrams for the given item.

Then similarity is computed between the given product and all other products, using SciKit Cosine Similarity.

Iterate through each item's similar items and store the 50 most-similar. Similarities and their scores are stored in database as a sorted set of the item.

Returns:None
